7. GROQ PAID TIER — COST PER USER CALCULATION
========================================================================

Model: Llama 3.3 70B Versatile
Input:  $0.59 per 1,000,000 tokens
Output: $0.79 per 1,000,000 tokens

--- Cost per API call (approximate) ---

  evaluateAnswer():
    ~1,500 input tokens × $0.59/M  = $0.000885
    ~1,500 output tokens × $0.79/M = $0.001185
    TOTAL: ~$0.002 per call

  evaluateDailyAnswer():
    ~1,200 input × $0.59/M  = $0.000708
    ~1,200 output × $0.79/M = $0.000948
    TOTAL: ~$0.0017 per call

  generateDailyQuestion():
    ~200 input × $0.59/M  = $0.000118
    ~200 output × $0.79/M = $0.000158
    TOTAL: ~$0.0003 per call

  Mock Interview (2 calls combined):
    ~3,500 input × $0.59/M  = $0.002065
    ~5,000 output × $0.79/M = $0.003950
    TOTAL: ~$0.006 per mock interview

--- Monthly cost per user type (30 days) ---

  FREE user (avg 2-3 calls/day):
    ~3 × $0.002 × 30 = ~$0.18/month

  PREMIUM user (avg 5-8 calls/day):
    ~7 × $0.002 × 30 + 1 mock/day × $0.006 × 30
    = $0.42 + $0.18 = ~$0.60/month

  PREMIUM user (maxed out every day):
    15 × $0.002 + 5 × $0.006 + $0.0003 = ~$0.06/day
    = ~$1.80/month ← worst case

--- Monthly cost for X daily active users (on Groq PAID) ---

  1,000 DAU (mixed free/premium):  ~$5-10/month
  5,000 DAU (mixed):               ~$25-50/month
  10,000 DAU (mixed):              ~$50-100/month
  50,000 DAU (mixed):              ~$250-500/month
